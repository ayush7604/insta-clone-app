# Instagram Mini Clone

This project is a mini Instagram-style application built to demonstrate backend development concepts such as authentication, database relationships, and REST APIs. A minimal frontend is included to interact with the backend.

## Tech Stack
- Node.js
- Express
- MongoDB
- JWT Authentication
- React (basic frontend)

## Features
- User signup and login
- Password hashing
- JWT-based authentication
- Follow and unfollow users
- Create posts with image URL and caption
- Like and comment on posts
- Feed showing posts from followed users

## Backend Setup
cd backend
npm install
node server.js

## Frontend Setup
cd frontend
npm install
npm start

## Notes
The frontend is intentionally minimal and is used only to demonstrate backend API functionality.
